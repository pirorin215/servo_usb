# Servo USB Controller

Pro MicroとSG90サーボモーターを使用したUSB制御スイッチシステム。赤外線送受信機能により、スマートリモコンとも連携可能です。

## 機能

- **サーボモーター制御**: SG90サーボモーターで物理的なスイッチ操作を自動化
- **オルタネートスイッチ対応**: ON/OFFの2状態をスムーズに切り替え
- **赤外線送受信**: スマートリモコンとの連携で遠隔操作が可能
- **HIDキーボード機能**: ディスプレイのウェイクアップ信号を送信
- **ブザー音フィードバック**: 操作状態を音で確認
- **低電力モード**: サーボをデタッチして消費電力を削減

## 必要なハードウェア

| コンポーネント |仕様|
|---|---|
| マイコン | Arduino Pro Micro (ATmega32U4) |
| サーボモーター | SG90 (または互換品) |
| スイッチ | オルタネートスイッチ |
| 赤外線LED | 赤外線送信用LED |
| 受信モジュール | VS1838Bまたは同等の赤外線受信モジュール |
| ブザー | 圧電ブザー |

## ピン接続

| Pro Microピン | 接続先 |
|---|---|
| D5 | サーボモーター信号線(オレンジ/黄) |
| D4 | オルタネートスイッチ |
| D3 | 赤外線LED (+抵抗直列) |
| D2 | 赤外線受信モジュール信号 |
| D6 | ブザー |
| VCC | サーボ赤線、スイッチ側 |
| GND | サーボ茶線、ブザー、受信モジュールGND |

### 回路図の注意点

- 赤外線LEDには必ず適切な抵抗（約100〜330Ω）を直列接続してください
- サーボモーターの電源はPro MicroのVCCから供給可能ですが、複数接続する場合は外部電源を検討してください

## 必要なライブラリ

Arduino IDEのライブラリマネージャから以下をインストールしてください：

- **Servo** (Arduino標準ライブラリ)
- **IRremote** by Armin Joachimsmeyer (v3.9.0以上)

## インストール手順

1. Arduino IDEを開く
2. `ツール` > `ボード` > `Arduino AVR Boards` > `Arduino Pro Micro` を選択
3. `ツール` > `シリアルポート` でPro Microのポートを選択
4. `servo_usb.ino` を開く
5. 必要なライブラリをインストール
6. 検証ボタン(✓)をクリック
7. アップロードボタン(→)をクリック

## 使い方

### 基本操作

1. **USB接続時**: ピッという短いブザー音が鳴り、起動を知らせます
2. **スイッチ操作**: 物理スイッチを操作すると、サーボが動作してスイッチを切り替えます
3. **赤外線受信時**:
   - ON信号受信: ピーー（長い音）
   - OFF信号受信: ピッピッ（2回の短い音）

### 赤外線信号フォーマット

- プロトコル: NEC形式
- 周波数: 38kHz
- アドレス: 0x00
- ONコマンド: 0x01
- OFFコマンド: 0x02

### シリアルモニタ

9600 baudでシリアルモニタを開くと、動作ログを確認できます：

```
START
A:90
Sw:H
L:1
READY
Act:1
IR sending:ON len:67 [...]
IR sent:ON
S:1
```

## スケッチのカスタマイズ

### サーボ角度の調整

```cpp
const Command OFF_COMMANDS[] = {
  {CMD_MOVE, 160},  // この角度を調整
  {CMD_WAIT, 800},
  {CMD_MOVE, 147},  // この角度を調整
  {CMD_DETACH, 0},
  {CMD_END, 0}

const Command ON_COMMANDS[] = {
  {CMD_MOVE, 80},   // この角度を調整
  {CMD_WAIT, 800},
  {CMD_MOVE, 90},   // この角度を調整
  {CMD_DETACH, 0},
  {CMD_WAIT, 1000},
  {CMD_KEY, WAKE_KEYCODE},
  {CMD_END, 0}
};
```

## トラブルシューティング

### サーボが動かない
- サーボの信号線が正しく接続されているか確認してください
- 電源が不足している場合、外部電源の使用を検討してください

### 赤外線受信が反応しない
- 受信モジュールの配線を確認してください
- 赤外線LEDが正しく接続されているか確認してください

### HIDキーボードが動作しない
- Pro Microを一度再接続してください
- シリアルモニタで「HID sent」と表示されているか確認してください
